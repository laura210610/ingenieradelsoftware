hexadecimal

El sistema hexadecimal es un sistema de numeración en base 16 que utiliza los dígitos del 0 al 9 y las letras A a F. Se usa principalmente en informática para representar datos binarios de manera más compacta y legible. Es útil para mostrar direcciones de memoria, códigos de colores y para la depuración de software.
